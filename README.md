# Week4-Day6-
CRUD Mini Module

Student CRUD Mini Module

Project Description This is a simple Student Management System built using HTML, CSS, and JavaScript.

It demonstrates basic CRUD operations:

Create (Add Student)
Read (View Students)
Update (Edit Student)
Delete (Remove Student)
Features

Add new student records
View student list in table format
Edit existing student details
Delete student records
Simple and clean user interface
Technologies Used

HTML
CSS
JavaScript
Project Structure

index.html → Structure of the application
style.css → Styling of the application
script.js → CRUD logic and functionality
Purpose This project was built as a practice module to understand CRUD operations in web development.
